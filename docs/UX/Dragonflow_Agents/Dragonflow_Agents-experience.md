## Purpose

Help a Dragonflow developer or AI engineer answer: *"Is this agent — with this prompt, these tools, and this model — going to behave the way I want before I ship it into a production workflow?"*

## Audience

Internal users only: Dragonflow's own developers and AI engineers configuring and iterating on the agents that power downstream Dragonflow products. Low single-digit concurrent users. Used in a tight build-and-try loop, typically alongside the code editor and the target agent's eventual deployment surface (Slack, voice, browser automation, etc.).
